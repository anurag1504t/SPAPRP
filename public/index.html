About Us

Students’ Performance and Project Reporting Portal

Students’ academic success is evaluated by their performance in exams conducted by the institutes or Universities. SPAPRP evaluates students’ academic performance with a fuzzy logic based performance evaluation method to evaluate students’ final academic performance. This project has various features like it calculates overall result of each individual based on various factors, it stores projects done by every individual in each semester with project titles and short descriptions.

  • Result analysis with a fuzzy logic-based method.
  • Portal to the students’ to check their results online.
  • Act as an interface to share project titles with descriptions.

These are some important features of the SPAPR Portal. Click on Learn More for detailed Information.
